パソコンちぇっ君（Pasokon Chekkun）— かんたん操作ガイド
========================================================

製作者: Alpha Script
ホームページ: https://alphascript-kyoto.github.io/as-homepage/
リポジトリ: https://github.com/AlphaScript-kyoto/PC-Health-Checker


■ 起動方法
  ・「PC-Chekkun-*-portable.exe」をダブルクリック
  ・ウィンドウを閉じてもトレイ（通知領域）に常駐します
  ・起動時に健康診断＋容量マップ作成を行い、設定した毎日の時刻にも自動スキャンします
    （設定タブの「毎日の自動スキャン時刻」。既定は 09:00）


■ 温度・SMART詳細を見るには（重要）
  Windows の仕様上、温度・通電時間・属性テーブルは
  【管理者権限】が必要です。

  次のいずれかで起動してください:
  1. 同梱の「run_as_admin.bat」を使う
  2. アプリ起動後、「管理者として再起動」
  3. exe を右クリック →「管理者として実行」

  UAC の確認が出たら「はい」を選んでください。


■ 画面の見方
  ホーム … 全体の状態と気になる項目
  ディスク … SMART・温度・通電時間・属性テーブル
  容量マップ … フォルダ容量の可視化と安全性ラベル
  提案 … 交換候補の目安とショップ検索リンク
  価格 … チェックしたパーツの週次価格記録（価格.com / Amazon / Keepa）
  ニュース … PC パーツ関連ニュース
  設定 … 通知・毎日の自動スキャン時刻・Windows起動時の自動起動など


■ データの保存場所
  %LOCALAPPDATA%\PCHealthChecker\data
  （クラウドへの送信はありません。この PC 内だけに保存します）


■ 自動購入について
  商品の自動購入機能はありません。
  リンクを開いて自分で購入する方式です。


■ うまく動かないとき
  ・すでに起動中ならトレイから一度「終了」してから開き直す
  ・SMART が空なら管理者として起動し直す
  ・ファイアウォール等で 127.0.0.1:8787 を遮断していないか確認
