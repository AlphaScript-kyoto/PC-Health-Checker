PC Health — かんたん操作ガイド
================================

製作者: Alpha Script
ホームページ: https://alphascript-kyoto.github.io/as-homepage/
バージョン: 0.2.4


■ 起動方法
  ・「PCHealth-*-portable.exe」をダブルクリック
  ・ウィンドウを閉じてもトレイ（通知領域）に常駐します
  ・起動時に1回スキャンし、設定した毎日の時刻にも自動スキャンします
    （設定タブの「毎日の自動スキャン時刻」。既定は 09:00）


■ 温度・SMART詳細を見るには（重要）
  Windows の仕様上、温度・通電時間・属性テーブルは
  【管理者権限】が必要です。

  次のいずれかで起動してください:
  1. 同梱の「run_as_admin.bat」を使う
  2. アプリ起動後、トレイ右クリック →「管理者として再起動」
  3. exe を右クリック →「管理者として実行」

  UAC の確認が出たら「はい」を選んでください。


■ 画面の見方
  ホーム … 全体の状態と気になる項目
  ディスク … 各ドライブの SMART・温度・通電時間
  提案 … 交換候補の目安とショップ検索リンク
  価格 … チェックしたパーツの週次価格記録（価格.com / Amazon / Keepa）
  ニュース … PC パーツ関連ニュース
  設定 … 通知・毎日の自動スキャン時刻・Windows起動時の自動起動など


■ データの保存場所
  %LOCALAPPDATA%\PCHealth\data
  （クラウドへの送信はありません。この PC 内だけに保存します）


■ 自動購入について
  商品の自動購入機能はありません。
  リンクを開いて自分で購入する方式です。


■ 価格取得の既知の不具合・注意点
  ・Amazon に短時間で連続アクセスすると一時的にブロックされ、
    「Amazon価格抽出失敗」や取得漏れが起きることがあります。
    時間を置いて「今すぐ価格更新」を押すと回復します。
  ・価格は検索結果ベースの参考値です。検索語によっては
    別商品やアクセサリの価格が混ざることがあります
    （外れ値は自動で除外しています）。
  ・Amazon の価格は「検索上位の新品出品の平均」です。
    実際のカート価格と異なる場合があります。
  ・Keepa グラフは検索結果の先頭商品のものを表示するため、
    意図と違う商品のグラフになることがあります
    （グラフ上部の商品名で確認できます）。
  ・価格.com / Amazon のページ構成が変わると
    取得できなくなる場合があります。


■ うまく動かないとき
  ・すでに起動中ならトレイから一度「終了」してから開き直す
  ・SMART が空なら管理者として起動し直す
  ・ファイアウォール等で 127.0.0.1:8787 を遮断していないか確認
